## Installation
- Download this repository.
or clone the repo :
```
git clone https://github.com/bimbims125/mazer-django.git
```
- Make virtual environment
```
py -m venv (your own name)
```
- Activate the Virtual Environment
```
env\scripts\activate
```

- Install django
```
pip install django
```
- run 
```
py manage.py runserver
```
